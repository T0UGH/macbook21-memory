# Plugins

## Installed by MT

- Dataview
- Obsidian Git

## Recommended next install in UI

- Calendar

## Why this set

- **Dataview**：后续做索引页、项目列表、研究列表很有价值
- **Obsidian Git**：适合把这个 vault 当成长期知识仓库持续提交
- **Calendar**：配合 Daily 使用体验更好，但本次没有强行自动安装

## Notes

如果 Obsidian 首次打开后没有自动启用 community plugins，可以在设置里确认社区插件已允许，然后检查：
- `.obsidian/community-plugins.json`
- `.obsidian/plugins/`
